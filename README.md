# UT1 – Lenguajes Scriptings
## Windows – Scripts en PowerShell
Ubicación: ASO/UT1_Lenguajes_scriptings/Windows/
Aquí se encuentran varios scripts realizados en PowerShell para practicar:
- Variables, tipos de datos y operadores.
- Entrada y salida de datos con el usuario.
- Condicionales, bucles (foreach, while, for).
- Arrays y manejo de ficheros.
Hay ejercicios para obtener tablas de multiplicar, suma de números o comprobar si un número es par o impar que he usado para comprender el funcionamiento de los comandos de PowerShell.
## Linux – Scripts en Bash
Ubicación: ASO/UT1_Lenguajes_scriptings/Linux/
En esta parte se incluyen varios scripts para practicar conceptos básicos de Bash, como:
- Condicionales (por ejemplo, Validar-numero.sh, Mayor-Edad.sh).
- Bucles (Imprimir-Secuencia.sh, While.sh).
- Funciones (Permisos.sh).
- Gestión de archivos y directorios (Detección-carpetas.sh).
Hay ejercicios para obtener “Hola mundo”, sumas, contar archivos o comprobar el estado de una web con los que he entendido como funcionan los scrips en sistemas Unix.
# UT2 – Directorios
Ubicación: ASO/UT2_Directorios/
En esta unidad hay varios documentos en PDF con actividades relacionadas con la Administración de Servicios de Directorio:
- Organización y estructura de archivos y directorios.
- Permisos y propiedad de los mismos.
- Comandos de administración básicos.
- Automatización mediante scripts.
